# 健康配餐网页

这是一个健康饮食网页应用，包含随机健康三餐、AI 营养分析、健康等级奖章、高糖饮品方糖可视化、冰箱食材搭配、AI 一周菜谱、健康日历、连续打卡段位、拍照识别和联网商品营养查询。

## 本地打开

```bash
npm start
```

然后打开：

```text
http://127.0.0.1:4187
```

## 发布到 GitHub Pages

把这些文件上传到仓库根目录并覆盖旧文件：

- `index.html`
- `styles.css`
- `app.js`
- `README.md`
- `package.json`
- `preview-server.js`
- `vercel.json`

GitHub Pages 选择 `main` 分支和 `/root` 目录即可。

## 当前功能

- 减脂、增肌、塑型三种目标
- 随机生成早餐、中餐、晚餐
- 大量常见原材料和组合菜谱
- AI 营养分析：热量、糖分、蛋白质、脂肪、钠含量
- AI 替代推荐：奶茶、可乐、炸鸡、汉堡等高风险食物的低糖低油选择
- 健康等级系统：S+ 到 D 细分等级，带动态奖章
- AI 一周菜谱：支持学生党、训练日、上班族、控糖模式和预算估算
- 健康日历：早餐/中餐/晚餐拆分记录，运动按有氧/无氧和时间自动估算消耗
- 段位统计：近一天、近一周、近一月、近一季度、近一年健康指数
- 打卡系统：连续打卡、补救期、等级徽章和鼓励动画
- 首次进入欢迎页和功能切换渐入动画
- 高糖饮品雷达：全球热门饮品方糖可视化
- 冰箱模式：输入已有食材和克数，生成健康菜谱
- 拍照识别：移动端拍照/上传、图片线索识别、包装条码识别入口、联网商品营养查询
- 联网营养查询：可按商品名或条形码查询公开食品数据库，并同步到 AI 健康助手
